Explanation of Important Tags

1. The <header> tag is used to display the title and introduction of the webpage.

2. The <main> tag contains the primary content of the webpage.

3. The <section> tag organizes related content into meaningful sections.

4. The <span> tag highlights important text within a paragraph.

5. The <img> tag is used to display an image that supports the content.

Accessibility Improvement

An alt attribute was added to the image so that screen readers can describe the image to users with visual impairments.
