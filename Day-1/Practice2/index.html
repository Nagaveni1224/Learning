HTML Important Tags

Explanation of Important Tags

1. The <header> tag is used to create the top section of the webpage.

2. The <main> tag is used to store the main content of the webpage.

3. The <section> tag is used to divide the content into different parts.

4. The <h1> and <h2> tags are used for headings and better content structure.

5. The <p> tag is used to write paragraphs and display text clearly.

Accessibility Improvement

I used semantic tags like header, main, and section to improve accessibility and help screen readers understand the webpage structure easily.
