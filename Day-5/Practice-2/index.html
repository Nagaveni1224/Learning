Why Important Tags Were Used

<header>

The header tag introduces the portfolio website. It contains the user's name and professional title. It helps visitors understand the purpose of the page. It is usually displayed at the top of the webpage. It improves the overall structure of the document.

<article>

The article tag is used for individual projects. Each project can be treated as independent content. It makes the code more organized. Search engines can better understand project information. It improves semantic meaning.

<ul> and <li>

These tags are used to create lists of skills. They present information in a clean format. Users can quickly scan the listed items. They improve readability. They help maintain a structured layout.

<img>

The image tag displays a profile picture. It makes the portfolio more attractive. It helps visitors recognize the owner of the portfolio. Images can support personal branding. The tag includes alternative text for accessibility.

Accessibility Improvement

Added clear and descriptive alt text to all images. This helps screen-reader users understand image content. It also provides information when images cannot be loaded.
