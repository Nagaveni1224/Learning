Tags Used Explaintion

  • <header> was used to display the website title and introduction. It helps visitors understand the purpose of the page immediately.
  • <main> was used to contain the primary content of the webpage. This improves semantic structure and accessibility.
  • <section> was used to organize content into Cricket, Basketball, and Volleyball categories.
  • <article> was used for each sports news story because each article can stand independently.
  • <footer> was used to display copyright information and provide a clear ending to the page.

Accessibility Improvement

Meaningful alt text was added to every image so screen readers can describe the images to users with visual impairments.
